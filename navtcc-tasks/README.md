🎓 NAVTTC — AI/ML Training Tasks
This repository contains a comprehensive collection of tasks and quizzes completed during the NAVTTC AI/ML Training Program at NETSOL & NIAI. The curriculum covers the journey from Python fundamentals to advanced Deep Learning architectures.

📚 Curriculum Overview
The tasks are organized into the following learning modules:

1. Python Foundations (Tasks 01–27)
Environment Setup & Linux Basics.

Control Flow: Conditional logic, loops, and jump statements.

Data Structures: Lists, Tuples, Sets, and Dictionaries.

Functional Programming & Exception Handling.

2. Data Science Essentials (Tasks 28–50)
NumPy: N-Dimensional arrays, slicing, reshaping, and mathematical operations.

Pandas: Series, DataFrames, cleaning real-world data, and correlation analysis.

Statistics: Mean, Median, Mode, and Standard Deviation for ML.

3. Machine Learning (Tasks 51–54)
Linear Regression & Decision Trees.

Support Vector Machines (SVM).

Model Evaluation using Scikit-Learn.

4. Deep Learning & Computer Vision (Tasks 56–64)
Neural Networks from scratch (MLP, Feedforward).

Understanding Loss Functions.

CNNs: Image classification using CIFAR-10 and architectural concepts.

5. Natural Language Processing (Tasks 60–62)
Basics of NLP and Text Processing.

Advanced NLP using the NLTK library.

🛠️ Tech Stack
Programming: Python

Libraries: NumPy, Pandas, Scikit-Learn, NLTK, Matplotlib, Seaborn

Environment: Jupyter Notebooks, Google Colab
